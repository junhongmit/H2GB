Heterophilic and Heterogeneous Datasets

MAGDataset

The ogbn-mag and modified mag-year dataset from the "Open Graph Benchmark: Datasets for Machine Learning on Graphs" paper.

OAGDataset

A variety of new heterogeneous graph benchmark datasets composed of subsets of Open Academic Graph (OAG) from "OAG: Toward Linking Large-scale Heterogeneous Entity Graphs" paper.

RCDDDataset

The risk commodity detection dataset (RCDD) from the "Datasets and Interfaces for Benchmarking Heterogeneous Graph Neural Networks" paper.

IeeeCisDataset

IEEE-CIS-G is a heterogeneous financial network extracted from a tabular transaction dataset from IEEE-CIS Fraud Detection Kaggle Competition.

PokecDataset

H-Pokec is a heterogeneous friendship graph of a Slovalk online social network, collected from SNAP at Stanford University.

PDNSDataset

PDNS is a heterogeneous cybersecurity graph of passive DNS data from the "PDNS-Net: A Large Heterogeneous Graph Benchmark Dataset of Network Resolutions for Graph Learning" paper.
